# Random Items Shop (DOM Practice Project)

## Overview
This is a simple frontend project built to practice **JavaScript DOM manipulation** and basic web development concepts. The page displays a list of random items for sale and allows users to add items to a cart while dynamically updating the total price.

The goal of this project is to strengthen understanding of how JavaScript interacts with the **Document Object Model (DOM)** to create dynamic web pages.

---

## Features

- Display a list of items dynamically
- Add items to a cart
- Update total price automatically
- Basic layout using CSS
- Interactive buttons using JavaScript event listeners

---

## Technologies Used

- HTML5

- DOM Manipulation

## How It Works

The items are stored in a JavaScript array. Using DOM methods such as:

- `document.getElementById()`
- `document.createElement()`
- `appendChild()`
- `addEventListener()`

The page dynamically generates item cards and updates the cart when a user clicks the **Add to Cart** button.
